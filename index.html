clifa

Pretty online train connection api client for your shell written in python. Based on pyefa.

A list of supported regions can be found here.

Also supports location aliases and optaining location via wifi.

Install

Just install it from the Python Package Index:

sudo pip install clifa

Update

sudo pip install -U clifa

Install from source

The Python-module pyefa is required.

Just clone the repository and run:

python setup.py install

Usage

# For all options and complete usage, see:
clifa -h
clifa route -h
# and so on

# get connections from Düsseldorf Hbf to Essen Hbf
clifa route düsseldorf hbf essen hbf

# set alias "omnom" for essen Hbf
clifa alias add omnom essen hbf

# get connections from Düsseldorf Hbf to Essen Hbf (by alias)
clifa route düsseldorf hbf omnom

# get connections from Essen Hbf (by alias) to Düsseldorf Hbf
clifa route --from omnom --to düsseldorf hbf

# set default network to vrr
clifa network vrr

# associate current wifi with address Sibyllastraße 9 in Essen
clifa wifi add Essen "addr:Sibyllastraße 9"

# from current wifi to address via Oberhausen Hbf
clifa alias add chaosdorf Düsseldorf "addr:Hüttenstr. 25"
clifa route --to chaosdorf --via Oberhausen Hbf
